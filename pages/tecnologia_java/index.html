Tecnologia

Nome: Java

Aniversário: 1995

Site Oficial: https://www.java.com/pt-BR/

Java é uma linguagem de programação orientada a objetos, de propósito geral, que segue o princípio "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar), significando que o código Java pode ser executado em qualquer dispositivo ou sistema operacional, desde que tenha a Máquina Virtual Java (JVM) instalada. Essa característica se deve à separação entre a linguagem Java e a plataforma subjacente, permitindo que o código seja compilado em bytecode que é interpretado pela JVM. Java é amplamente usado para o desenvolvimento de aplicativos web, móveis (especialmente Android), sistemas corporativos, e muito mais, devido à sua robustez, segurança e portabilidade.
