Tecnologia

Nome: JavaScript

Aniversário: 1995 por Brendan Eich

Site Oficial: https://www.javascript.com

JavaScript é uma linguagem de programação de alto nível, dinâmica, fraca e interpretada, amplamente utilizada no desenvolvimento web. Originalmente, foi criado para adicionar interatividade a páginas da web, sendo executado no lado do cliente (navegador), mas com o advento de tecnologias como o Node.js, passou a ser utilizado também no lado do servidor. JavaScript permite a manipulação do DOM (Document Object Model) para modificar elementos em páginas web, fazer requisições assíncronas e criar funcionalidades interativas, tornando as páginas mais dinâmicas e responsivas. É uma das três tecnologias principais da web, juntamente com HTML e CSS. Sua popularidade se deve à sua flexibilidade, ampla adoção e suporte em todos os navegadores modernos.

Prós

  1. Execução no Lado do Cliente
  2. Desenvolvimento Full-Stack com Node.js
  3. Grande Comunidade e Ecossistema

Limitações

  2. Desempenho (em comparação com linguagens compiladas)
  3. Gerenciamento de Código Assíncrona
