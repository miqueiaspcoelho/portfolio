Tecnologia

Nome:SQLite

Aniversário: 2000 por D. Richard Hipp

Site Oficial: https://www.sqlite.org

SQLite é um sistema de gerenciamento de banco de dados relacional (SGBDR) leve, embutido e de código aberto. Ele é diferente de outros bancos de dados relacionais tradicionais, pois não utiliza um servidor de banco de dados separado. Em vez disso, o banco de dados SQLite é armazenado em um único arquivo de disco, tornando-o extremamente compacto e fácil de implementar. SQLite é amplamente utilizado em aplicações móveis, em sistemas embarcados, e em dispositivos com recursos limitados, devido à sua simplicidade e portabilidade. Ele é frequentemente escolhido para aplicações que necessitam de um banco de dados local, de pequeno porte, mas sem a complexidade de configurar e gerenciar um servidor de banco de dados completo.

Prós

  1. Leveza e Simplicidade
  2. Portabilidade
  3. Zero Configuração

Limitações

  1. Escalabilidade Limitada
  2. Conectividade Limitada
  3. Falta de Funcionalidades Avançadas
